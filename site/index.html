Privacidade

Sem backend obrigatório no MVP.

Scanora prioriza processamento local, armazenamento local e compartilhamento por APIs padrão do Android. O app não exige login nem sincronização em nuvem para funcionar.

  • OCR no dispositivo com ML Kit
  • Exportação de PDF, JPG e PNG localmente
  • Permissão de câmera apenas para captura manual
  • Sem upload automático de documentos

Instalação

Como rodar o projeto.

git clone https://github.com/Soturine/scanora.git
cd scanora
./gradlew assembleDebug

Use Android Studio com JDK 17+ e Android SDK Platform 36 com Build Tools 36.0.0.

GitHub Pages: após publicar o repositório, ative em Settings > Pages > Source: GitHub Actions.

FAQ

Perguntas rápidas.

Funciona offline?

Sim. O MVP prioriza processamento local. O scanner guiado pode depender do Google Play services no aparelho.

O CI está ativo?

Sim. O Android CI builda o projeto e roda lint e testes unitários no GitHub Actions.

Já está pronto para produção?

É uma base séria e funcional, mas ainda faltam capturas reais, QA visual e lapidação de release.
